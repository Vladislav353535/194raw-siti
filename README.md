Сайт публикуется через GitHub Pages с использованием GitHub Actions.

- Настроены минимальные права для GitHub Actions.
- В проект добавлен базовый файл .gitignore для исключения временных файлов.
- Рекомендуется включить защиту ветвей и требовать ревью для изменений в main.
